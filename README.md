# 📞 Call Center Management System (C++)

## 🚀 Overview

This is a simple **Call Center Management System** built using C++.
It simulates how customer support teams handle incoming calls, track issues, and resolve them.

The project demonstrates core **C++ programming concepts**, including Object-Oriented Programming (OOP) and STL.

---

## 🎯 Features

* ➕ Add new customer calls with issue details
* 📋 View all customer calls
* ✅ Mark calls as resolved
* ❌ Handle invalid inputs safely

---

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL - `vector`)
* Console-based interface

---

## 🧠 Concepts Implemented

* Classes and Objects (OOP)
* Encapsulation
* Dynamic data storage using `vector`
* Input handling using `getline`, `cin.ignore`, and `cin.fail()`
* Basic CRUD operations

---

## ▶️ How to Run

### Step 1: Compile

```bash
g++ main.cpp -o app
```

### Step 2: Run

app
